# 神经网络

## 相关概念

### 1. 线性函数

线性函数是神经网络的基础，形式为：`f(x) = wx + b`

- **w (weight)**：权重，控制输入对输出的影响程度
- **b (bias)**：偏置，调整输出的基准位置
- **线性组合**：多层神经网络的本质是多次线性变换的复合

### 2. 损失函数

损失函数用于衡量模型预测结果与真实值之间的差距。

**常见损失函数：**
- **均方误差 (MSE)**：用于回归问题
  ```
  L = 1/n Σ(y_pred - y_true)²
  ```
- **交叉熵损失**：用于分类问题
- **目标**：通过训练最小化损失函数，提高模型准确性

### 3. 激活函数

激活函数引入非线性，使神经网络能够学习复杂的模式。

**常见激活函数：**

- **ReLU (Rectified Linear Unit)**
  ```
  f(x) = max(0, x)
  ```
  - 优点：计算简单，缓解梯度消失
  - 广泛用于隐藏层

- **Sigmoid**
  ```
  f(x) = 1 / (1 + e^(-x))
  ```
  - 输出范围 (0, 1)
  - 常用于二分类输出层

- **Softmax**
  - 将输出转换为概率分布
  - 用于多分类问题

### 4. 梯度下降

梯度下降是一种优化算法，用于通过迭代调整参数来最小化损失函数。

**核心思想：**
1. 计算损失函数对参数的梯度（偏导数）
2. 沿梯度的反方向更新参数
3. 公式：`θ = θ - α∇L(θ)`
   - θ：参数
   - α：学习率（learning rate）
   - ∇L(θ)：梯度

**变体：**
- **批量梯度下降**：使用全部数据计算梯度
- **随机梯度下降 (SGD)**：每次使用一个样本
- **小批量梯度下降**：折中方案，使用小批量数据

### 5. 反向传播

反向传播算法是训练神经网络的核心方法，用于高效计算梯度。

**工作流程：**
1. **前向传播**：输入数据经过各层计算，得到输出
2. **计算损失**：比较预测输出与真实标签
3. **反向传播**：从输出层向输入层反向传播误差
   - 应用链式法则计算梯度
   - `∂L/∂w = ∂L/∂y × ∂y/∂w`
4. **参数更新**：使用梯度下降更新各层权重

**链式法则：**
```
如果 y = f(u) 且 u = g(x)
则 dy/dx = dy/du × du/dx
```

## 神经网络结构

### 基本组成
- **输入层**：接收原始数据
- **隐藏层**：进行特征提取和转换
- **输出层**：产生最终预测

### 前馈过程
```
输入 → [线性变换 → 激活函数] → 隐藏层 → ... → 输出层 → 预测结果
```
