# 线性代数

线性代数是人工智能和机器学习的数学基础，主要研究向量空间和线性变换。

## 核心概念

### 1. 矩阵 (Matrix)

**定义**：矩阵是向量的线性变换，本质是坐标系的变换。

**几何理解**：
- 矩阵乘法可以看作是对空间进行变换
- 将向量从一个坐标系映射到另一个坐标系
- 变换包括：旋转、缩放、剪切等

**应用**：
- 神经网络中的权重矩阵
- 图像变换和处理
- 特征提取和降维

### 2. 行列式 (Determinant)

**定义**：行列式表示基向量围成的面积或体积的变换比例。

**几何意义**：
- **det > 0**：保持空间定向（不翻转）
- **det < 0**：空间定向翻转
- **det = 0**：空间被压缩，面积或体积变成 0
  - 意味着维度降低（如 3D → 2D）
  - 矩阵不可逆

**直观理解**：
```
2D 空间：det = 变换后平行四边形面积 / 原单位正方形面积
3D 空间：det = 变换后平行六面体体积 / 原单位立方体体积
```

### 3. 秩 (Rank)

**定义**：矩阵的秩表示矩阵的列空间（或行空间）的维度。

**几何意义**：
- **满秩**：矩阵变换保持空间维度不变
  - 例如：3×3 矩阵秩为 3，3D 空间变换后仍是 3D
- **降秩**：矩阵变换导致空间降维
  - 例如：3×3 矩阵秩为 2，3D 空间被压缩到 2D 平面
  - 此时 det = 0

**重要性质**：
- 一旦降维，det（面积或体积）直接变成 0
- 降维意味着信息丢失

### 4. 逆矩阵 (Inverse Matrix)

**定义**：对于一个 n×n 矩阵 A，如果存在矩阵 B 使得 AB = BA = I（单位矩阵），则称 B 为 A 的逆矩阵。

**可逆条件**：
- **满秩（det ≠ 0）**：矩阵可逆
- **不满秩（det = 0）**：矩阵不可逆（奇异矩阵）

**几何理解**：
- 可逆矩阵的变换可以被"撤销"
- 不可逆变换导致信息丢失，无法恢复

**应用**：
- 求解线性方程组
- 矩阵除法运算

### 5. 重要关系

行列式、可逆性、秩之间的关系：

```
行列式非零 ⟺ 矩阵可逆 ⟺ 秩为满秩 ⟺ 齐次方程组只有零解
```

### 6. 线性方程组

对于方程组 Ax = b：

**解的情况判定**：
- **满秩且方阵**：有唯一解
- **不满秩**：
  - 若 b 在列空间内：有无穷多个解
  - 若 b 不在列空间内：无解

**最小二乘解**：
当方程组无解时，可以找到使 ||Ax - b||² 最小的近似解

## 在机器学习中的应用

### 神经网络
- 权重矩阵实现特征的线性变换
- 多层矩阵变换实现复杂函数逼近

### 主成分分析 (PCA)
- 利用特征值分解进行降维
- 保留主要信息，减少计算量

### 语义向量化
- 词向量的矩阵变换
- 注意力机制中的矩阵运算
